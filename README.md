# PokéAR Battle (Blog Site)

This is a simple, clean blog-style website generated from your PowerPoint content.

## Run locally
Open `index.html` in your browser.

## Deploy to GitHub Pages (easy)
1) Create a GitHub repo (public is easiest).
2) Upload these three files: `index.html`, `styles.css`, `script.js`
3) In GitHub: Settings → Pages → Build and deployment:
   - Source: Deploy from a branch
   - Branch: `main` / root
4) Your site will appear at: https://<username>.github.io/<repo>/

## Customize
- Replace the hero mockup with screenshots from your Unity build.
- Add a `gallery` section if you want multiple images.
